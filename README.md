# BusinessLens AI

AI-Powered Business Analyst Assistant built for Hackathon 2026 using Gemini API, Streamlit, Pandas and Plotly.

BusinessLens AI transforms raw CSV business datasets into interactive dashboards, visual analytics and AI-generated business insights within seconds.

---

# Project Overview

BusinessLens AI is an intelligent business analytics platform designed to help users quickly understand sales performance, market trends and business opportunities through automated data analysis and Generative AI.

Users can upload a CSV dataset and instantly receive:

- KPI metrics
- Interactive visualizations
- Correlation analysis
- Data quality analysis
- AI-powered strategic insights

---

# Features

## Smart CSV Upload

Upload business-related CSV datasets directly into the platform.

## Interactive Dashboard

Automatically generated KPI metrics including:

- Total Sales
- Total Orders
- Average Sales
- Top Performing Country

## Visual Analytics

Dynamic visualizations powered by Plotly:

- Sales Distribution Histogram
- Top Countries by Sales
- Product Line Sales Share
- Correlation Matrix

## Gemini AI Insights

Integrated Gemini API generates business-focused strategic insights such as:

- Market opportunities
- Product performance evaluation
- Risk analysis
- Expansion recommendations

## Data Exploration

- Missing value detection
- Descriptive statistics
- Numeric and categorical column identification

# How to Run Locally

## Clone Repository

```bash
git clone https://github.com/Busrataseven/businesslens-ai.git
```

---

## Install Requirements

```bash
pip install -r requirements.txt
```

---

## Create .env File

```env
GEMINI_API_KEY=YOUR_API_KEY
```

---

## Run Application

```bash
streamlit run app.py
```
